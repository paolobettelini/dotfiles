# dotfiles

My personal dotfiles for Arch Linux.

+ [sxhkd](./sxhkd) key-binder for X11
+ [bspwm](./bspwm) tiling window manager for X11
+ [fish](./fish) fish shell
+ [xinit](./xinit) X server launcher

Other dependencies

+ `Picom` compositor
+ `rofi` application launcher
+ `firefox` browser
+ `alacritty` terminal
